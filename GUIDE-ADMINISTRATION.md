# Guide d'administration - Site Yoyo'bar

Ce guide vous explique comment gérer le contenu de votre site web via l'interface d'administration Strapi.

---

## 1. Accéder à l'interface d'administration

1. Ouvrez votre navigateur
2. Accédez à : `http://localhost:1337/admin` (ou l'URL de votre serveur)
3. Connectez-vous avec vos identifiants

---

## 2. Gérer les cocktails

### Ajouter un nouveau cocktail

1. Dans le menu de gauche, cliquez sur **"Cocktail"**
2. Cliquez sur le bouton **"Créer une nouvelle entrée"** (en haut à droite)
3. Remplissez les champs :
   - **Nom** (obligatoire) : Le nom du cocktail
   - **Description** : Une courte description des ingrédients
   - **Prix** : Le prix en euros (ex: 12.50)
   - **Catégorie** : Choisissez parmi Cocktail, Mocktail, Bière, Vin, Soft
   - **Image** : Cliquez sur "Ajouter un média" pour télécharger une photo
4. Cliquez sur **"Enregistrer"**
5. Cliquez sur **"Publier"** pour rendre le cocktail visible sur le site

### Modifier un cocktail

1. Cliquez sur **"Cocktail"** dans le menu
2. Cliquez sur le cocktail à modifier
3. Faites vos modifications
4. Cliquez sur **"Enregistrer"**

### Supprimer un cocktail

1. Cliquez sur **"Cocktail"** dans le menu
2. Cliquez sur le cocktail à supprimer
3. Cliquez sur la corbeille rouge en haut à droite
4. Confirmez la suppression

---

## 3. Gérer les événements

### Ajouter un événement

1. Dans le menu, cliquez sur **"Evenement"**
2. Cliquez sur **"Créer une nouvelle entrée"**
3. Remplissez les champs :
   - **Titre** (obligatoire) : Le nom de l'événement
   - **Date** (obligatoire) : La date et l'heure de l'événement
   - **Description** : Détails de l'événement
   - **Image** : Photo ou affiche de l'événement
   - **A la une** : Cochez si vous voulez mettre en avant cet événement
4. **Enregistrer** puis **Publier**

### Conseils pour les événements

- Ajoutez les événements à l'avance pour que vos clients puissent les voir
- Les événements passés restent visibles dans l'onglet "Passés" du site
- Utilisez des images attractives pour attirer l'attention

---

## 4. Gérer la galerie photos

### Ajouter des photos

1. Cliquez sur **"Image Galerie"** dans le menu
2. Cliquez sur **"Créer une nouvelle entrée"**
3. Remplissez les champs :
   - **Image** (obligatoire) : Téléchargez votre photo
   - **Légende** : Une courte description (optionnel)
   - **Date** : La date de la photo/soirée (optionnel)
4. **Enregistrer** puis **Publier**

### Conseils pour les photos

- Utilisez des photos de bonne qualité
- Évitez les photos floues ou mal cadrées
- Variez les types de photos (ambiance, cocktails, soirées, équipe...)

---

## 5. Modifier les informations du bar

1. Cliquez sur **"Informations Bar"** dans le menu
2. Modifiez les informations :
   - **Adresse** : L'adresse complète du bar
   - **Téléphone** : Numéro de téléphone
   - **Email** : Adresse email de contact
   - **URL Instagram** : Lien vers votre page Instagram
   - **URL Google Maps** : Lien vers votre localisation Google Maps
   - **Texte de présentation** : Le texte "À propos" du bar
   - **Image Hero** : L'image principale de la page d'accueil
3. **Enregistrer** puis **Publier**

---

## 6. Bonnes pratiques

### Images

- **Format recommandé** : JPEG ou PNG
- **Taille recommandée** :
  - Cocktails : 400x500 pixels
  - Événements : 600x400 pixels
  - Galerie : 800x600 pixels minimum
- **Poids maximum** : 2 Mo par image

### Textes

- Gardez les descriptions courtes et percutantes
- Vérifiez l'orthographe avant de publier
- Utilisez des mots-clés pertinents pour le référencement

### Publication

- **Brouillon** : Le contenu n'est pas visible sur le site
- **Publié** : Le contenu est visible par tous les visiteurs
- Vous pouvez dépublier un contenu à tout moment

---

## 7. Résolution de problèmes

### L'image ne s'affiche pas sur le site

1. Vérifiez que le contenu est bien **publié**
2. Vérifiez le format de l'image (JPEG, PNG)
3. Rechargez la page du site (Ctrl+F5)

### Le contenu ne s'affiche pas

1. Vérifiez que le statut est bien "Publié"
2. Attendez quelques secondes et rafraîchissez la page
3. Si le problème persiste, contactez le développeur

### J'ai oublié mon mot de passe

Contactez le développeur pour réinitialiser votre mot de passe.
